Android SDR Recon

Urchin tracks TPMS traffic, not Bluetooth.

Urchin is a local-first Android app for capturing TPMS observations from rtl_433. It supports RTL-SDR USB capture plus network-fed rtl_433 JSON, including HackRF-backed feeds, then keeps the sightings, raw payloads, and sensor history on-device.

Download Debug APK CLI Build Steps
USB Capture RTL-SDR USB capture now relays Android's granted device fd into bundled rtl_433.
Bridge Mode Point the app at any TCP endpoint streaming newline-delimited rtl_433 JSON.
Local Store Sensor list, sightings, raw JSON export, and diagnostics stay on the device.

Features

  • TPMS protocol parsing with vendor lookup hints
  • Live, starred, and battery-low filters
  • Signal-sorted history and raw JSON export
  • Diagnostics with recent application logs

Hardware

  • RTL2832U / RTL-SDR USB dongles
  • HackRF-hosted rtl_433 bridge over TCP
  • Remote rtl_433 bridge over TCP
  • 315 MHz and 433.92 MHz presets

Focus

  • No Bluetooth permissions
  • No BLE enrichment or alert rules
  • No cloud sync or remote collection
  • Just SDR capture, TPMS history, and export
